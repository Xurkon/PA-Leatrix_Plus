**Leatrix Plus** is a comprehensive quality of life addon for World of Warcraft 3.3.5a, ported for **Project Ascension**. It includes a ton of features to automate tasks, enhance the UI, and streamline your gameplay experience.

![Leatrix Plus Interface](https://github.com/user-attachments/assets/84067d46-3ece-4f9e-a5fa-001961310a23)

## ✨ Features

### 🤖 Automation
- **Faster Looting** - Significantly speeds up the looting process
- **Auto Quest Accept/Turn-in** - Automatically handles quest interactions
- **Auto Sell Junk** - Sells grey items at vendors
- **Auto Repair** - Repairs gear automatically at vendors
- **Auto Accept Summons** - Handles summon requests
- **Auto Release in PvP** - Quick release in battlegrounds

### 💬 Chat Enhancements
- Hide combat log and chat buttons
- Unclamp chat frame for free positioning
- Move chat editbox
- Class colors in chat
- Recent chat window with copy/paste
- Increase chat history
- Chat message filtering

### 🎨 Interface Improvements
- **Minimap Enhancements** - Square minimap, combine addon buttons, resize
- **Tooltip Customization** - Enhanced tooltips with additional info
- **Quest Log Improvements** - Better quest tracking
- **Error Frame Control** - Smaller/faster error messages
- **Flight Map Enhancements** - Show all flight paths

### 🛡️ Social Controls
- Block duels, party invites, friend requests
- Auto-accept from friends/guild members
- Manage unwanted interactions

## 📦 Download & Installation

1. **[Download](https://github.com/Xurkon/PA-Leatrix_Plus/releases/latest)** the latest release
2. Extract the `Leatrix_Plus` folder to `Interface\AddOns\`
3. Replace/overwrite any existing files

## 🎮 Usage

| Command | Action |
|---------|--------|
| `/ltp` | Open configuration menu |
| Minimap Click | Open config |
| Minimap Shift+Click | Reload UI |
| Minimap Alt+Click | Toggle error messages |
