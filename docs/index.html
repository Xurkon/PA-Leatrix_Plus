📖 Overview

PA-Leatrix_Plus is a comprehensive quality of life addon for World of Warcraft 3.3.5a, specifically ported and maintained for Project Ascension. Originally created by Leatrix and ported to WoTLK 3.3.5 by Sattva-108, this version includes fixes and compatibility patches for Project Ascension's unique client modifications.

The addon is modular by design - you only enable the features you want, keeping your UI clean and your game running efficiently. Configuration is done through an intuitive in-game menu accessible via /ltp or the minimap button.

✨ Features

Leatrix Plus includes a massive collection of quality of life improvements organized into categories:

🤖 Automation

Faster looting, auto quest accept/turn-in, auto sell junk, auto repair, auto accept summons, and auto release in PvP.

💬 Chat Enhancements

Hide chat buttons, unclamp chat frame, class colors, copy/paste support, increased history, and message filtering.

🗺️ Minimap

Square minimap option, combine addon buttons, resize minimap, and enhanced tracking.

💡 Tooltips

Enhanced tooltip information, customizable positioning, and additional item details.

📜 Quest Log

Improved quest tracking, quest level display, and quest automation options.

✈️ Flight Paths

Show all available flight paths, enhanced flight map, and taxi improvements.

⚠️ Error Frame

Smaller error messages, faster fade time, and custom inventory full sounds.

🛡️ Social Controls

Block duels, party invites, friend requests. Auto-accept from friends and guildmates.

📦 Installation

  1. Download the latest release from GitHub Releases
  2. Extract the Leatrix_Plus folder
  3. Copy the folder to your WoW Interface\AddOns\ directory
  4. Restart World of Warcraft or type /reload

📁 Addon Location

For Project Ascension, the typical path is:
Ascension Launcher\resources\client\Interface\AddOns\Leatrix_Plus\

🎮 Usage

Access the configuration menu using any of these methods:

Action Result
/ltp Open configuration menu
Minimap Button Click Open configuration menu
Minimap Button Shift+Click Reload user interface
Minimap Button Alt+Click Toggle error messages visibility

Configuration Tips

  • Options are organized into pages - use the navigation at the bottom of the config window
  • Some options require a UI reload - the "Reload" button will glow when needed
  • Hover over options to see detailed tooltips explaining each feature
  • Use Shift+Ctrl+Click on some config buttons for preset profiles

🔧 Ascension-Specific Fixes

This port includes fixes specifically for Project Ascension's modified 3.3.5a client:

InterfaceOptionsControlsPanelAutoLootCorpse Fix

Original Error

attempt to index global 'InterfaceOptionsControlsPanelAutoLootCorpse' (a nil value)

Problem: The addon was trying to access a standard WoW UI element that doesn't exist in Project Ascension's modified client.

Solution: Added nil checks before accessing the element:

if InterfaceOptionsControlsPanelAutoLootCorpse then
    InterfaceOptionsControlsPanelAutoLootCorpse:Disable()
    local autoLootText = InterfaceOptionsControlsPanelAutoLootCorpseText
    if autoLootText then
        autoLootText:SetText("Auto Loot option is controlled by Leatrix Plus.")
        autoLootText:SetAlpha(0.6)
    end
end

The auto-loot functionality still works through the SetCVar calls - it just won't modify the (non-existent) Interface Options checkbox.
